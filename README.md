# **Gallery**
Незатейливая фотогалерея на ReactJs с возможностью добавлять альбомы, изображения, обрабатывать их фильтрами и скачивать на свое устройство.

В проекте используются:

* Семантические теги
* Адаптивная вёрстка
* Flex-контейнеры
* Canvas 
* BEM 
* Относительные величины
* и многое другое
